Text Formatting Tags

Italic - This makes the sentence italic.

Bold - This makes the sentence bold.

Mark tag - This is used for highlighting.

Delete tag - It strikes out the text.

Subscript - The text appears half a character below the normal line. Example: H2O

Superscript - The text appears half a character above the normal line. Example : 22 =4

Stong - Displays in bold but has semantic meaning

Underline - The text isunderlined.

Insert - The text is insertedin the sentence.

Line break - Adds a line break in the sentence.
It is a empty tag.

Horizontal rule- Adds a horizontal line.
